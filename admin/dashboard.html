📦 Import Student Data

Students can export their progress data using the Export My Data button in their account menu (top-right corner of any page). This creates a JSON file that contains all their progress, enrollments, and certificates.

Use case: Testers can work offline with a copy of this app, export their data, and send it back. You can then import it here to merge their progress into the main system.

  • Student profile and name
  • Course enrollments
  • Progress in each module
  • Quiz results and scores
  • Certificates earned

Getting Started

Welcome to QA Pilot Academy

Let's get your platform configured in 4 quick steps.

This wizard will walk you through the core setup so you can start adding students and tracking progress right away. You can come back to any of these settings at any time from the Settings tab.

Platform Settings

Set your instructor email so students can send progress reports directly to you.

Students submit progress reports you can import to the dashboard.

Set your instructor email so the student export button pre-fills an Outlook email addressed to you. Set an organisation name to brand the platform for your cohort.

Both fields are optional — you can skip and set them later in Settings → Platform Settings.

⚠️ The default password is QAAdmin2026. Change it before sharing this platform with students.

You can skip this step and change the password later in Settings → Admin Password.

You're all set! The next step is to add your students.

How the student flow works:
  1. Add a student here → set their name, email, role, and assigned courses.
  2. Use Generate Setup File to create a .json file for them.
  3. Email the file to the student via Outlook.
  4. Student opens the login page, uploads the file → instant account setup.
  5. When done, student clicks Export Progress → Outlook pre-fills with their results addressed to you.
  6. You import the file here to see their grades and certificates.

Click Add First Student below to open the student creation form, or close this wizard and explore the dashboard first.

Reset Entire Application

This will permanently delete everything:

  • All student accounts and their progress
  • All course enrollments
  • All settings (admin password, page wait time, content customizations)
  • All bug toggles
  • All certificates

Courses will be restored to their factory defaults.

This action cannot be undone. All data will be lost.
